# 🌿 GraSS — База рабочих инструкций

Корпоративное веб-приложение для централизованного хранения рабочих инструкций, гайдов и памяток. Проект создан для того, чтобы заменить хаотичное хранение документов в чатах и папках единой, удобной базой знаний.

## 🚀 Основные возможности

* **Управление контентом (CRUD):** Создание, чтение, обновление и безопасное удаление инструкций.
* **Работа с файлами:** Прикрепление к инструкциям скриншотов, PDF-документов, текстовых файлов и ZIP-архивов (ограничение размера — до 20 МБ).
* **Встроенный просмотр:** Загруженные изображения автоматически отображаются в теле инструкции, а документы выводятся в виде кнопок для скачивания.
* **Умный поиск и фильтрация:** Поиск инструкций по названию и содержимому, а также фильтрация по рабочим категориям (Принтеры, ТСД, Сеть, Windows, 1С и др.).
* **Корпоративный стиль:** Адаптивный и аккуратный интерфейс в фирменных зелено-белых цветах с использованием Bootstrap.

## 🛠 Стек технологий

* **Язык и фреймворк:** C#, ASP.NET Core MVC
* **База данных:** SQLite (интеграция через Entity Framework Core)
* **Вёрстка:** HTML, CSS, Razor (`.cshtml`), Bootstrap 5
* **Архитектура:** Model-View-Controller (MVC)

## 📁 Структура проекта

* `Controllers/` — Логика приложения (обработка запросов, поиск, сохранение файлов).
* `Models/` — C#-классы базы данных (`Instruction`, `Attachment`) и контекст Entity Framework (`AppDbContext`).
* `Views/` — HTML-шаблоны страниц приложения.
* `wwwroot/uploads/` — Локальное хранилище загруженных пользовательских файлов и скриншотов.
* `app.db` — Локальный файл базы данных SQLite.
